Sprint 1

Group Goal: to setup the backbone of the application and create basic features

User story I worked on:
  • As an app admin, I want an upload button to easily upload new data file and update vendor info
Task completed:
  • Configured RPC mechanism for loading data remotely

Sprint 2

Group Goal: to incorporate more essential features to our VanFood App

User stories and tasks I worked on:
  • As a user I want to be able to add and remove food vendor’s to or from a list of favorites, to keep it updated as my tastes change
    • create user class to store the favorites list for each user
    • create a new tab for vendor table to hold favourites table
  • As a user, I want to be able to access my favourites list so I can keep a record of what I like
    • create “add to favourites” button for each vendor in the favourites column
    • implement function to add favorited vendors to favorites table
    • create the favourites table with the same functions as the vendors table

Key things I learned

  • How to effectively practice version control with teammates on GitHub
  • How to do acceptance testing
  • How to successfully persist data with GAE datastore for saving users' favourites
